Como Funciona

Este sistema se desarolla en un enfoque de facil manejo, este sistema mostrara reportes estadisticos desarrollados en pentaho en base a consultas sql para ser mostrados facilmente al usuario en una pagina web sin la necesidad de otro software.

Sotware Utilizado

  • Eclipse
  • PostgresQL
  • Pentaho data integration
  • Pentaho report designer
  • Java
  • Repositorio en GitHub

FUENTES

Los datos en los que se manejan nuestras estadisticas son tomados del banco de datos libres que se encuentran en la plataforma del INEC
